Tools, hardware, and software I use to build and ship apps.

## Editor & Terminal

- **Claude Code** — primary development tool, multi-agent workflows
- **VS Code** — secondary editor for quick edits and file browsing
- **Xcode** — SwiftUI development and iOS builds
- **iTerm2 + zsh** — terminal

## Languages & Frameworks

- **TypeScript** — primary language across web and mobile
- **Swift** — native iOS development
- **Python** — backend scripts and tooling
- **Expo / React Native** — cross-platform mobile apps
- **SwiftUI** — native iOS UI
- **Next.js** — web applications

## Backend & Data

- **Supabase** — auth, database, edge functions
- **Turso** — edge SQLite for low-latency reads
- **Cloudflare Workers** — edge compute and API proxies

## Mobile

- **EAS Build** — cloud builds for iOS
- **TestFlight** — beta testing distribution
- **RevenueCat** — in-app purchases and subscriptions
- **Expo Router** — file-based navigation

## AI Tools

- **Claude Code** — multi-agent development workflows
- **Claude API** — vision and text capabilities in apps

## Infrastructure

- **GitHub** — source control and collaboration
- **Vercel** — web app hosting
- **Cloudflare** — DNS, workers, edge network
- **GitHub Pages** — static sites (including this one)

## Design

- **Figma** — occasional wireframing and mockups
- **SF Symbols** — iOS iconography
- **AI-generated app icons** — prompt-based icon design

## Hardware

- **MacBook Pro** — M-series, daily driver
- **iPhone** — real-device testing
